# Cpp_project_3

## Задание: Сделать программу «Битва динозавров».

### Функционал:

- Покупка, хранение, выбор для боя динозавров;
- Создание противника игрока, у которого пул случайных дино в каждом бою;
- Сражение динозавров c отображением в консоли дино и
пошаговые изменения HP;
- Автоматическое сохранение всех изменений в файле;
- Загрузка данных из файла при открытии приложения.
